Forking Strategy

A "fork" is creating two separate winning threats simultaneously. Your opponent can only block one. Set up forks by placing marks that create multiple potential winning lines. This advanced tactic wins games.

Block Opponent's Forks

When opponent threatens to create a fork, you must block it immediately. Sometimes blocking obvious threats leaves fork opportunities—learn to see them coming.

With perfect play from both sides, Tic Tac Toe always ends in a draw. If you're losing, aim for the draw by blocking all threats. If you're winning, set up forks your opponent can't stop.

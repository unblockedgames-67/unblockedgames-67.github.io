Game Modes And Mechanics

Geometry Dash features multiple gameplay modes that change how you control your character [citation:10]:

Cube Mode

The default mode. Tap to jump, release to fall. Simple but requires perfect timing. Used in most levels for basic platforming sections.

Ship Mode

Press and hold to fly upward, release to fall. Gravity-defying sections require fine control. Hold Spacebar continuously to maintain height, tap for small adjustments.

Ball Mode

Gravity flips each time you tap. Up becomes down, down becomes up. Disorienting but rhythmic once you learn the pattern.

UFO Mode

Tap to jump upward while maintaining horizontal momentum. Like the cube but with more floaty physics. Great for obstacle sequences.

Wave Mode

Two-dimensional movement—tap to move up, release to move down. Navigate through narrow corridors and spike mazes.

Robot Mode

Variable jump height based on how long you hold. Tap lightly for small hops, hold for big jumps. Adds precision to platforming.

Spider Mode

Teleports between floor and ceiling with each tap. Instant movement requires anticipating where you'll land.
